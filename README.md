# 💰 المحاسب الشخصي - تطبيق تيليجرام المصغر (Cloud Version)

تطبيق ويب مصغر (Telegram Mini App) متطور لإدارة الحسابات المالية الشخصية، مصمم بواجهة عصرية تعتمد على نمط **Glassmorphism** ويدعم المزامنة السحابية عبر خوادم تيليجرام.

## ✨ المميزات الرئيسية (Key Features)

* ☁️ **المزامنة السحابية (Telegram Cloud Storage):** يتم حفظ بياناتك وحساباتك تلقائياً في حسابك على تيليجرام، مما يتيح لك الوصول إليها من أي جهاز آخر.
* 💵 **دعم العملات المتعددة:** إدارة أرصدتك بثلاث عملات أساسية (الدولار $، الليرة التركية ₺، الليرة السورية ل.س).
* 📊 **كشوفات حساب تفصيلية:** عرض تاريخ العمليات المالية لكل حساب على حدة داخل التطبيق مع ملخص للأرصدة.
* 📄 **تصدير PDF:** إمكانية توليد تقارير مالية احترافية بصيغة PDF لكل حساب بضغطة زر واحدة.
* 🛡️ **نظام حذف آمن:** ميزة حذف الحسابات مع "تطهير" تلقائي لكافة الحركات المالية المرتبطة بها لضمان دقة الأرصدة الإجمالية.
* 🎨 **واجهة مستخدم احترافية:** تصميم عصري (Dark Mode) متجاوب بالكامل مع الهواتف الذكية وتأثيرات زجاجية جذابة.
* 🔔 **ردود فعل لمسية (Haptic Feedback):** اهتزازات خفيفة تفاعلية عند حفظ العمليات أو الحذف لتحسين تجربة المستخدم.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5 & CSS3:** الهيكل الأساسي مع تصميم مخصص.
* **Tailwind CSS:** لتنسيق الواجهة بسرعة واحترافية.
* **JavaScript (ES6+):** لمعالجة المنطق البرمجي والعمليات الحسابية.
* **Telegram Web App API:** للربط مع ميزات تيليجرام مثل الحفظ السحابي والتنبيهات.
* **html2pdf.js:** لتحويل تقارير HTML إلى ملفات PDF قابلة للتحميل.

## 🚀 طريقة التشغيل (Setup & Deployment)

1.  **استضافة الملف:** قم برفع ملف `index.html` (أو `accountant_pro.html`) على أي استضافة تدعم HTTPS (مثل GitHub Pages، Vercel، أو Netlify).
2.  **إنشاء البوت:**
    * اذهب إلى [@BotFather](https://t.me/botfather) في تيليجرام.
    * قم بإنشاء بوت جديد باستخدام أمر `/newbot`.
    * قم بتفعيل ميزة الـ **Web App** للبوت عبر خيار `Bot Settings` > `Menu Button` أو `Direct Link`.
3.  **ربط الرابط:** ضع رابط الملف المستضاف في خانة رابط تطبيق الويب الخاص بالبوت.
4.  **الاستخدام:** ابدأ باستخدام البوت مباشرة من داخل تطبيق تيليجرام.

## 🔒 الأمان والخصوصية

* التطبيق يعمل بالكامل في بيئة تيليجرام الآمنة.
* لا يتم إرسال بياناتك المالية إلى أي خوادم خارجية؛ البيانات تُخزن حصراً في **Telegram Cloud Storage** المخصص للمستخدم.
